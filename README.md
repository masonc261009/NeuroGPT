NeuroGPT 

NeuroGPT is a simple, local AI assistant that can read your files and answer questions about them.

It works with things like text files and Excel spreadsheets, helping you quickly understand and use your own data.

 Features
   -Read files from your filesystem
   -Ask questions about your data
   -Basic Excel (.xlsx, .csv) support
   -Uses AI to understand and summarize content
   -Runs locally (your files stay on your machine)
 
 How It Works
 
  -Loads files from a chosen folder
  -Extracts and formats the data
  -Sends relevant content to an AI model
  -Returns a clear answer to your question

 Frontend

NeuroGPT includes a simple UI built with MudBlazor for a clean and responsive experience.



 Supported Files
  .txt
  .csv
  .xlsx

 Notes
 
  Only reads files from the folder you choose
  Does not modify files unless you add that feature
